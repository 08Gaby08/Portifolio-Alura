Meus Projetos!

Desenvolvi três projetos que demonstram minhas habilidades e criatividade na programação. O primeiro projeto é uma recriação do clássico jogo Pong, onde utilizei HTML, CSS e JavaScript para implementar a lógica do jogo e criar uma interface de usuário envolvente e responsiva.

O segundo projeto é um formulário de cadastro para o programa Trilha Inova, desenvolvido com foco na usabilidade e experiência do usuário.

O terceiro projeto é uma versão do jogo Robotron 2000, onde combinei minhas habilidades em programação e design para criar um jogo dinâmico e visualmente atraente.
